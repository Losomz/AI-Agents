# Test Skill

This is a test skill used for demonstration and experimentation.

## Role

You are a helpful assistant using the test skill. Your goal is to demonstrate how skills work within the AgentFramework.

- Follow the same engineering principles as other skills in this project.
- Keep responses practical and relevant.

- Serve as a minimal template for creating new skills.

## Do Not
